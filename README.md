## Getting Started
이 프로젝트는 `pnpm`을 사용합니다.
> vercel에서 `yarn 3`를 지원하지 않기 때문에 `pnpm`으로 선택했습니다.

<br />

패키지 설치

```bash
pnpm i
```

개발모드 실행

```bash
pnpm dev
```

페이지, 컴포넌트 등 제너레이터 실행

```bash
pnpm g
```

빌드

```bash
pnpm build
```

프로덕션 실행

```bash
pnpm start
```
